# SOL-ACT 기능 목록

> 작성일: 2026-04-04
> 버전: Production

---

## 사용자 역할

| 역할 | 설명 |
|------|------|
| 학생 (STUDENT) | 수업 확인, 과제 제출, 연습 영상/일지 업로드, 식단 기록 |
| 선생님 (TEACHER) | 수업 관리, 과제 부여/채점, 평가, 학생 관리 |
| 원장 (DIRECTOR) | 전체 관리, 클래스 생성, 사용자 관리, 초대코드 발급 |

## 1. 인증

| 기능 | 설명 | 대상 |
|------|------|------|
| 회원가입 | 초대코드 기반 역할별 가입 | 모든 사용자 |
| 로그인 | 이메일 + 비밀번호 | 모든 사용자 |
| 비밀번호 찾기 | 이름 + 이메일로 임시 비밀번호 발급 | 모든 사용자 |

---

## 2. 대시보드 (홈)

### 학생
- 오늘의 수업 목록 (시간, 장소, 과목)
- 진행 중인 과제 수
- 오늘 식단 칼로리
- D-Day 카운트 (다가오는 대회)
- 빠른 버튼: 연습영상 올리기, 연습일지 쓰기

### 선생님/원장
- 전체 학생/선생님 수
- 오늘 전체 수업 수
- 미제출 과제 수
- 최근 활동 피드 (과제 제출, 일지 작성, 포트폴리오 등록 등)

| 영상 업로드 | 복수 영상 동시 선택/업로드, 5MB 청크 × 3 병렬 전송 | 학생 |

## 13. 파일 업로드 인프라

| 기능 | 설명 |
|------|------|
| 청크 업로드 | 10MB 이상 파일 자동 분할 (5MB × 3 병렬) |
| 이어올리기 | 중단된 업로드 자동 재개 (localStorage) |
| 영상 압축 | 서버에서 자동 (H.264, 720p, CRF 28) |
| 썸네일 생성 | 영상 1초 지점 자동 캡처 |
| 외장하드 지원 | SSD 연결 시 자동 감지, 미연결 시 로컬 저장 |
| 화면 유지 | 업로드 중 Wake Lock + "화면 유지" 안내 |
| 최대 크기 | 영상 1.5GB, 문서 50MB |

## 기술 스택

| 구분 | 기술 |
|------|------|
| 프론트엔드 | React 19, TypeScript, Tailwind CSS |
| 백엔드 | FastAPI (Python), SQLAlchemy, SQLite |
| 실시간 | WebSocket |
| 푸시 | Web Push (VAPID), pywebpush |
| 영상 처리 | FFmpeg (서버 압축 + 썸네일) |
| 파일 저장 | 로컬 + 외장 SSD (자동 감지) |
| 배포 | ngrok 터널 |
