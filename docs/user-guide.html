로그인

가입한 이메일과 비밀번호로 로그인합니다.

  • 이메일과 비밀번호를 입력하고 로그인 버튼을 누릅니다.
  • 비밀번호를 잊었다면 비밀번호 찾기를 눌러 임시 비밀번호를 받으세요.

회원가입

처음 사용하는 경우 회원가입 탭으로 전환합니다.

  • 학원에서 받은 초대 코드를 입력합니다.
  • 코드 확인 후 역할(수강생/선생님)이 자동으로 표시됩니다.
  • 이름, 이메일, 비밀번호(8자 이상, 영문+숫자+특수문자)를 입력합니다.
  • 비밀번호 규칙이 실시간으로 ✓ 표시됩니다.
초대 코드는 1회용입니다. 한 번 사용하면 다시 사용할 수 없습니다.

  • 진행 중인 과제 — 아직 제출하지 않은 과제 수

캘린더 & 수업 목록

  • 캘린더에서 파란 점이 있는 날짜에 수업이 있습니다.
  • 날짜를 탭하면 해당 날짜의 수업 목록이 표시됩니다.
  • 수업 카드를 탭하면 상세 화면으로 이동합니다.
  • 상세 화면에서 수업일지를 작성하거나, 선생님은 출석 체크를 할 수 있습니다.

수업일지 작성

  • 수업 상세 화면에서 수업일지 탭을 선택합니다.
  • 오늘 배운 점이나 느낀 점을 기록합니다.
  • 사진, 영상, 음성 파일을 첨부할 수 있습니다.

개인 레슨 신청 (학생)

  • 보라색 개인 레슨 신청 버튼을 누릅니다.
  • 선생님, 과목, 희망 일시, 사유를 입력합니다.
  • 신청 상태(대기중/승인/거절)를 확인할 수 있습니다.
선생님은 수업 등록, 수업 완료 처리, 출석 체크를 할 수 있습니다. 개인 레슨 신청이 오면 승인/거절 패널이 표시됩니다.

포트폴리오

연습 영상을 기록하고 성장 과정을 관리합니다.

  • 그리드 보기: 썸네일로 전체 영상을 한눈에 확인
  • 타임라인 보기: 연습 시리즈별로 시간순 정리
  • 영상을 탭하면 재생 화면으로 이동합니다.

영상 업로드

  • + 등록 버튼을 누릅니다.
  • 제목, 카테고리(독백/장면연기/뮤지컬/즉흥연기/오디션 준비), 태그를 입력합니다.
  • 영상 파일을 선택합니다 (10분 이내 권장, 최대 1.5GB).
  • 업로드 진행률이 표시되며, 완료 후 서버에서 자동으로 영상을 최적화합니다.
  • 이미 등록된 포트폴리오에 "영상없음"이 표시된 경우, 업로드 버튼을 눌러 영상을 추가할 수 있습니다.

영상 재생

  • 영상을 탭하면 재생됩니다.
  • 재생 바를 드래그하여 원하는 지점으로 이동할 수 있습니다.
  • 선생님은 특정 시간에 타임스탬프 코멘트를 남길 수 있습니다.
영상은 업로드 후 서버에서 자동으로 압축됩니다. 원본 화질을 유지할 필요 없이 편하게 올려주세요.
선생님은 학생별/카테고리별로 필터링하여 포트폴리오를 확인하고, AI 피드백을 요청하거나 직접 코멘트를 남길 수 있습니다.

선생님이 작성한 정기 평가를 확인합니다.

식단 캘린더

  • 캘린더에서 식단이 기록된 날짜에 점이 표시됩니다.
  • 날짜를 탭하면 해당 날짜의 식단 목록과 총 칼로리를 볼 수 있습니다.

식단 기록하기

  • 초록색 식단 기록 버튼을 누릅니다.
  • 식사 유형(아침/점심/저녁/간식)을 선택합니다.
  • 식단 내용을 입력합니다 (예: "김치찌개, 밥, 계란후라이").
  • 음식 사진을 첨부할 수 있습니다.
  • 분석 요청을 누르면 AI가 칼로리를 추정하고 영양 조언을 해줍니다.
선생님은 전체 학생의 식단을 목록으로 보고, 각 식단에 코멘트를 남길 수 있습니다.

푸시 알림

앱 설치 (PWA)

  • 홈 화면에 추가를 누르면 일반 앱처럼 사용할 수 있습니다.
  • iOS: 하단 공유 버튼 → "홈 화면에 추가"
  • Android: 브라우저 메뉴 → "앱 설치" 또는 "홈 화면에 추가"
