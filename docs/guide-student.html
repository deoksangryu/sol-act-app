접속 방법

  • 학원에서 안내받은 주소(URL)를 브라우저에 입력합니다

  • 학원에서 받은 초대 코드를 입력하면 "수강생"으로 자동 설정됩니다

포트폴리오 (연습 영상)

  • + 등록으로 새 포트폴리오를 만듭니다
  • 제목, 카테고리(독백/장면연기/뮤지컬/즉흥연기/오디션 준비), 영상을 등록합니다
  • 그리드: 썸네일로 전체 보기 / 타임라인: 시리즈별 시간순 보기
  • 영상을 탭하면 재생되며, 재생 바로 원하는 지점 이동 가능
  • "영상없음" 카드에서 업로드 버튼으로 영상 추가

평가 리포트

  • 선생님이 작성한 정기 평가를 확인합니다
  • 연기력, 표현력, 창의성, 협동심, 성실도 5개 항목 점수
  • 과목별(연기/뮤지컬/무용) 필터 가능

대회·행사

  • 다가오는 대회/오디션의 D-Day를 확인합니다
  • 일시, 장소, 접수 기간 정보를 볼 수 있습니다
  • 체크리스트로 준비 상황을 관리합니다 (예: 대본 준비, 의상 준비 등)
영상은 업로드 후 서버에서 자동 압축됩니다. 원본 그대로 올리세요. 업로드 중에는 앱을 닫거나 다른 페이지로 이동하지 마세요.
