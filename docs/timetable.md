## 데이터 모델 요약

```
ClassInfo.schedule: List[{
  day: str,            # '월'|'화'|'수'|'목'|'금'|'토'|'일'
  start_time: str,     # 'HH:mm'
  subject: str,        # 'acting'|'musical'|'dance'|'basics'
  subject_label: str,  # 사람이 읽는 과목명
  teacher_id: str,     # users.id
  location: str        # '부평'|'산곡'
}]
```

- `subject_teachers`: `{과목키: teacher_id}` — 강사가 본인 화면에서 자신의 반을 조회하는 데 사용.
- 수업 종료 시간은 DB에 저장되지 않고 시작+2h로 계산.
- `generate_lessons_for_class`: 내일부터 4주치 정규 수업 레코드를 자동 생성.
