수업 등록 & 관리

  • 수업 등록: 클래스, 날짜, 시간, 과목, 장소를 선택하여 수업을 만듭니다
  • 수업 카드를 탭하면 상세 화면으로 이동합니다
  • 상세 화면에서 수업 완료 / 수정 / 삭제 처리
  • 수업 상태는 자동 변경되지 않습니다:
    • 예정 → 수업 생성 시 기본 상태
    • 완료 → 선생님이 직접 "수업 완료" 버튼을 눌러야 변경됩니다
    • 취소 → 선생님이 직접 취소 처리해야 합니다

수업일지 & 출석

  • 수업 상세의 수업일지 탭에서 수업 내용, 목표, 다음 계획을 기록합니다
  • 사진/영상/음성 파일을 첨부할 수 있습니다
  • 출석 탭에서 학생별 출결을 체크합니다:
    • 출석 / 지각 / 결석 / 사유결석
    • 사유가 있으면 메모를 남길 수 있습니다

평가 작성

  • + 평가 작성으로 학생 정기 평가를 등록합니다
  • 학생, 클래스, 과목, 기간을 선택합니다
  • 5개 항목(연기력/표현력/창의성/협동심/성실도) 점수(1~5)
  • 코멘트를 작성합니다 (학생에게 보여집니다)
  • 작성 후 수정/삭제 가능

포트폴리오 관리

  • 학생별 필터로 포트폴리오 영상을 확인합니다
  • 영상에 타임스탬프 코멘트를 남길 수 있습니다 (특정 시간 지점에 피드백)

대회·행사 관리

  • 대회/오디션 일정을 등록합니다 (행사명, 일시, 장소, 접수 기간)
  • 체크리스트로 준비 상황을 관리합니다

클래스 채팅

  • 하단 소통 탭에서 클래스별 채팅, 질의응답, 공지사항을 관리합니다
  • 학생 질문에 답변하거나 공지를 작성합니다

식단 코멘트

  • 학원 관리 → 식단 탭에서 전체 학생 식단을 확인합니다
  • 학생별 필터로 특정 학생의 식단만 볼 수 있습니다
  • 각 식단에 코멘트를 남겨 영양 조언을 합니다

칭찬 스티커

  • 학원 관리 → 칭찬 탭에서 학생에게 스티커를 보냅니다
  • 12가지 이모지(⭐🏆💪🎭🎵💃❤️🔥👏🌟🎯📈) 중 선택
  • 응원 메시지를 함께 보낼 수 있습니다
  • 학생은 프로필에서 받은 스티커를 확인합니다

학원 관리 (원장 전용)

  • 클래스 탭: 클래스 생성/수정, 담당 선생님 배정, 학생 배정
  • 사용자 탭: 전체 사용자 목록 확인, 정보 수정
학원 관리 메뉴는 하단 네비게이션의 마지막 탭(관리)에서 접근합니다.
