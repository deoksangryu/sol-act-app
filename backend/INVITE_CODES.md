# 인증코드 시스템 사용 가이드

## 개요

회원가입 시 학원에서 발급한 **인증코드**를 입력해야만 가입할 수 있습니다.
인증코드에 역할(수강생/선생님)이 지정되어 있어, 코드 입력 시 역할이 자동으로 설정됩니다.

## 인증코드 생성 (서버 PC에서 실행)

```bash
cd backend

# 수강생 코드 1개 생성
python generate_codes.py --role student

# 수강생 코드 5개 생성
python generate_codes.py --role student --count 5

# 선생님 코드 생성 (메모 포함)
python generate_codes.py --role teacher --count 2 --memo "3월 신규 강사"

# 특정 학생용 코드 생성 (메모로 구분)
python generate_codes.py --role student --memo "홍길동"
```

### 출력 예시

```
========================================
  인증코드 3개 생성 완료 (수강생)
========================================
  AB3K7HNP
  R4DWXM2Y
  J6FG8QTV
========================================
```

## 코드 목록 조회

```bash
cd backend

# 전체 코드 목록
python generate_codes.py --list

# 미사용 코드만 조회
python generate_codes.py --list --unused
```

### 출력 예시

```
코드          역할      상태      사용자        메모                  생성일
--------------------------------------------------------------------------------
  AB3K7HNP   수강생    미사용                  홍길동               2025-03-05 14:30
  R4DWXM2Y   수강생    사용됨    s1a2b3c4                          2025-03-05 14:30
  J6FG8QTV   선생님    미사용                  3월 신규 강사         2025-03-05 14:28

총 3개
```

## 회원가입 절차 (학생/선생님)

1. 학원에서 인증코드를 전달받음 (카카오톡 등)
2. SOL-ACT 앱 → 회원가입 탭 선택
3. **인증코드 입력** → 유효하면 역할이 자동으로 표시됨 (초록색)
4. 이름, 이메일, 비밀번호 입력
5. 회원가입 완료

- 인증코드는 **1회만 사용** 가능합니다.
- 코드가 유효하지 않으면 나머지 입력 필드가 비활성화됩니다.

## 코드 형식

- 8자리 영문 대문자 + 숫자
- 혼동되기 쉬운 문자 제외 (O/0, I/1)
- 예: `AB3K7HNP`

## 관리 API (localhost 전용)

서버에서 직접 API로도 조회할 수 있습니다:

```
GET http://localhost:8000/api/admin/invite-codes
```
